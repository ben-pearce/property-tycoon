Class: Dice

Dice(game)

This class represents our dice.

Constructor

new Dice(game)

Creates three dice sprites, the hand roll sprite, dice one and two sprites for rolling animation. dice.result always stores the latest result but does not update until the dice have landed!
Parameters:
Name Type Description
game GameManager The game manager the Dice belongs to.
Properties:
Name Type Description
result Array.<integer> Stores the result for most recent dice roll.
rollSprite Phaser.GameObjects.Sprite The hand roll sprite.
diceOneSprite Phaser.GameObjects.Sprite
diceTwoSprite Phaser.GameObjects.Sprite

Example usage of events

let dice = new Dice(game);

Methods

clickRoll(pointer)

This is called when the rollSprite is clicked. Removes the active listeners from the sprites, and makes the two dice visible. Both dice are tweened to a random position and keep cylcing frames until they "land", at which point this.land() will be called.
