Class: Property

Property(scene, tile, color)

This class represents an upgradable property. Rentable tiles will reference this class to give players a visual representation and record of the property associated with the tile.

getValue() → {integer}

This returns the upgrade value of the property. Needed to calculate the total value of a property after upgrades. The value of the upgrades is calculated by multiplying the original cost of the property it belongs to by the number of upgrades made. (note this does NOT return the total value of the property, use Rentable.getValue() for that.)
