Class: CashText

UI.CashText(target, cash)

A special animated text for when players earn and lose cash. Text "floats up" from the target movieclip and then quickly fades away. This will destroy itself after animation completes.

Constructor

new CashText(target, cash)

Takes a target movieclip and integer cash value.
Parameters:
Name Type Description
target Phaser.GameObjects.Sprite The target movieclip to float from.
cash integer The cash value.
