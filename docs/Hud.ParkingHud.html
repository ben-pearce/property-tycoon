Class: ParkingHud

Hud.ParkingHud(scene, hud, parking)

This class represents the parking HUD object within the HUD layer. It shows the free parking icon a name and a cash value.

Constructor

new ParkingHud(scene, hud, parking)

Creates a hud background, graphic, name text and cash text for the hud layer to represent the free parking space.
Parameters:
Name Type Description
scene Phaser.Scene The scene this HUD belongs to.
hud Hud The parent hud object.
parking Parking The parking tile instance.
Properties:
Name Type Description
hud Hud The hud layer this belongs to.
parking Parking The parking tile this observes.
